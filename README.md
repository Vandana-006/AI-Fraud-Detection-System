# AI Fraud Detection System

This project uses Machine Learning to detect fraudulent credit card transactions.

The system analyzes transaction data and predicts whether a transaction is fraud or not using a Random Forest model.

## Dataset
- Credit Card Fraud Detection Dataset
- Total transactions: 284,807
- Features: 31
- Target variable:
  - 0 = Non Fraud
  - 1 = Fraud

## Features
- Data analysis using Pandas
- Data visualization using Matplotlib and Seaborn
- Fraud detection using Random Forest
- Model evaluation using Accuracy, Precision, Recall and F1 Score
- Confusion Matrix
- ROC Curve

## Results
Model Accuracy: **99.96%**

Fraud Detection Metrics:
- Precision: **94%**
- Recall: **82%**
- F1 Score: **87%**

- Seaborn
